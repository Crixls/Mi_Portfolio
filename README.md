# 💼 Portfolio Personal

Este es mi portfolio web donde muestro mis proyectos, habilidades y experiencia como desarrolladora. La página está diseñada para tener una buena experiencia visual, responsiva y fácil de navegar.

## 🚀 Contenido

* 🧑‍💻 Información personal y presentación
* 📁 Proyectos destacados
* 📬 Formulario o sección de contacto
* 🌐 Enlaces a redes profesionales

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* React
* Vite
* Tailwind CSS

## 🎯 Objetivo

El objetivo de este portfolio es mostrar mi evolución como desarrolladora y servir como espacio centralizado para mis proyectos y contacto profesional.

## 💻 Ejecución

- npm install
- npm run dev
